## 💡 Tips

### Best Practices:

1. **Database:**
   - Indexes regular update karo
   - ANALYZE monthly run karo
   - Monitor query times

2. **Code:**
   - useMemo jahan expensive calculations
   - useCallback for event handlers
   - Lazy load non-critical components

3. **Images:**
   - WebP/AVIF use karo
   - Right size for device
   - Lazy load below fold

4. **Caching:**
   - Static assets: Long cache
   - API data: No cache
   - Smart invalidation
